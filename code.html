The Code for Re-Verse is structured in 3 functions. 


                      function captureString() {
                        // Capture and store user string
                        const userString = document.getElementById('inputString').value;
                        document.getElementById('inputString').value = '';
                    
                        const revString = reverseString(userString);
                    
                        displayString(revString);
                      }
captureString()

The function captureString() captures the input from the user and stores it in a variable called userString. In order to make the input ready for the next string, we will clear the input field as well, so we insert an empty string.

Next we call the function reverseString() with the variable userString as a parameter, and store it's returned value in a new variable called revString.

Finally we'll call the function displayString() with the variable revString as a parameter so that we can display the newly reversed string on the page. 


                      function reverseString(userString) {
                        let revString = '';
                    
                        for (let i = userString.length - 1; i >= 0; i--) {
                          revString += userString[i];
                        }
                    
                        return revString;
                      }
reverseString()

The function reverseString() takes in the userString from the previous function as a parameter.

Next we declare an empty string called revString, where we will store our reversed string.

To reverse the string, we will run it through a for-loop where we iterate (i) through the length of the userString, minus 1. This is because we start at 0 when counting, not at 1. The loop will continue for as long as i is greater than or equal to 0, and we will iterate in reverse (--) until we're done.

Each time we iterate, we will concatenate (+=) the value (character) that is occupying the current index (i) of userString. Since we are doing this in reverse order, we will now eventually have a reversed string stored in the revString variable.

Finally, we return revString. Note that we do this outside the for-loop, after it is done iterating. 


                      function displayString(revString) {
                        document.getElementById('results').innerHTML = revString;
                      }
displayString()

The function displayString() takes in the revString variable we returned in the previous function as a parameter, and it only has one job.

We store the contents of the revString variable inside the HTML of the results ID on the page. This way, the reversed string will now display in the paragraph that contains this ID.
